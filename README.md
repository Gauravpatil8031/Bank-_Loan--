# Bank Loan Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes bank loan data to evaluate loan performance, approval trends, and default risk patterns.

## Business Objective
To help financial institutions make better lending decisions by identifying good vs bad loans, risk distribution, and funding trends.

## Key Highlights
- 38K Total Loan Applications
- 86% Good Loans | 14% Bad Loans
- $435M Total Funded Amount
- Loan analysis by State, Grade, Purpose & Employment Length

## Tools & Skills Used
- Power BI
- Data Modeling
- DAX Measures
- Data Visualization

## How to View
Download the .pbix file and open it using Power BI Desktop.
